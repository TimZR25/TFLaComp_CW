Оператор ввода READ(*,*)

Оператор используется для ввода данных с клавиатуры (стандартный ввод) без строгого указания формата. Данные интерпретируются автоматически на основе типов переменных в списке ввода.

Формат записи:

READ(*, *) variable1, variable2, ...;

Параметры:

  • Первый * — обозначает стандартный ввод (обычно клавиатура)
  • Второй * — свободный формат (данные читаются в зависимости от типа переменных)
  • variable1, variable2, ... — список переменных для записи значений
  • ; — завершает оператор

Примечание: FORTRAN нечувствителен к регистру: READ, read, ReAd — одинаковы.

Примеры корректных записей:

1) Чтение двух переменных:

READ(*, *) x, y;

2) Чтение массива:

read(*,*) arr;

3) Чтение переменных в разном регистре:

rEaD(*, *) var1, var2;

Особенность: В связи с разработанной автоматной грамматикой G[<R>] приведенные выше записи будут распознаны синтаксическим анализатором как корректные.
