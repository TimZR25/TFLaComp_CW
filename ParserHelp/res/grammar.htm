Формальная грамматика G[<R>] для оператора ввода READ(*, *)

Продукции P в нотации Хомского:

1. <R> → 'READ' <OB>

2. <OB> → '(' <FP>

3. <FP> → '*' <C>

4. <C> → ',' <SP>

5. <SP> → '*' <CB>

6. <CB> → ')' <VS>

7. <VS> → letter <VE>

8. <VE> → ',' <VS> | <E> | letter <VE> | digit <VE>

9. <E> → ';'

Терминальные символы:

letter → a | b | ... | z | A | B | ... | Z

digit → 0 | 1 | ... | 9

Составляющие грамматики G[<R>]:

• Z = <R>;

• VT = { (, ), *, ,, ;, a, b, ..., z, A, B, ..., Z, 0, 1, ..., 9};

• VN = {<R>, <OB>, <FP>, <C>, <SP>, <CB>, <VS>, <VE>};

Обозначения:

  • → - выводится в
  • | - альтернатива (или)
